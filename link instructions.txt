Run a command prompt as administrator.

Using mklink create a symbolic link from your Spring mods directory to the development directory.  The main reason for this is that we can't do things in the Programs File directory without administrator rights, and due to the virtual store system things might get a bit messy.

mklink /D [Your Spring Install Directory]\mods\darius.sdd [Your Project Directory]\src


To do the map(s) do about the same thing.

mklink /D [Your Spring Install Directory]\maps\testmap.sdd [Your Project Directory]\maps\testmap
